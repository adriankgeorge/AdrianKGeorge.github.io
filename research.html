I work at the intersection of ecology, the geosciences, and data visualization.

Current Projects

My Masters project involves developing Ice Age Mapper V2 (IAM) an interactive web map showing plant migration since the Last Glacial Maximum. I ran a user needs survey for IAM and will create a discussion activity for introductory earth systems Geography course including IAM.

Using Carto DL, I am creating workflows that will allow paleoecologists to create their own proportional symbol maps of paleoecological data from the Neotoma Paleoecology Database.
